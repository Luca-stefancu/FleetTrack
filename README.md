# FleetTrack
FleetTrack este o aplicație destinată gestionării flotei unei firme de transport. Scopul aplicației este de a facilita administrarea vehiculelor, a șoferilor și a curselor efectuate, oferind o evidență clară și organizată a activităților companiei.

Aplicația va permite gestionarea informațiilor despre autovehicule (număr de înmatriculare, marcă, model, capacitate de încărcare, stare tehnică), șoferi (date personale, permis de conducere, experiență) și curse (destinație, distanță, dată plecare/sosire, vehicul alocat, șofer responsabil).
